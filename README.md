## Run Locally

Clone the project

```bash
  git clone https://github.com/Harshal-Bhangale/harshal-dev.git
```

Go to the project directory

```bash
  cd folio-v1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
